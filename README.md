## Features

| Tab | What it does |
|-----|--------------|
| **Map Tracker** | Live overlay map with waypoints, landmarks, sanctuaries, and coordinate paste from clipboard |
| **Growth Tracker** | Per-dinosaur growth timers with stage breakdowns (Hatchling → Juvenile → Sub-Adult → Adult) |
| **Diet Reference** | Full diet slot data for all 20 Evrima species, sourced from EvrimaQuickGuide.com |
| **Nesting Guide** | Egg limits, nest types, and nesting mechanics at a glance |
| **Prime Checklist** | Track Prime Objective progress for your current dinosaur |
| **Performance Config** | Quick-reference for in-game graphics and performance settings |

## Download & Install

1. Go to the [**Releases**](https://github.com/XeTrinityz/IsleCompanion/releases/latest) page.
2. Download the `.exe` setup file under **Assets**.
3. Run the installer — no admin rights required for most setups.
4. Launch **Isle Companion** from the Start Menu or desktop shortcut.

> Updates are not automatic. Check this page for new releases.

---

## Requirements

- **Windows 10** (1903 or later) or **Windows 11**
- **Microsoft WebView2 Runtime** — already installed on most modern Windows systems. If the app fails to launch, [download it here](https://developer.microsoft.com/en-us/microsoft-edge/webview2/).
